## 3. Language Guidelines for Spanish Explanations

- **Términos técnicos**: Mantén en inglés los términos estándar (DataFrame, API, commit)
- **Verbos**: Usa infinitivos para instrucciones ("usar", "crear", "ejecutar")
- **Tono**: Cercano pero profesional, tutea al lector
- **Ejemplos**: Usa datos y contextos relevantes para audiencia hispanohablante

## 4. Quality Check

Before delivering, verify:
- [ ] Explanation progresses from simple to complex
- [ ] Technical terms are defined or kept in recognizable form
- [ ] Code examples are tested and work
- [ ] No assumptions beyond stated prerequisites
- [ ] Practical application is clear
