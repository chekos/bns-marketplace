## Important Reminders

When using these templates:

1. **Always quote specific passages** to justify scores
2. **Be calibrated** - 3 is competent, 4+ requires justification
3. **Prioritize actionable feedback** - tell the writer what to DO
4. **Connect weaknesses** - show how fixing one helps others
5. **Acknowledge strengths genuinely** - specific praise, not generic
6. **Match depth to context** - quick reads need quick feedback
7. **Focus on high-leverage changes** - what moves the needle most?
