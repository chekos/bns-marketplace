$ARGUMENTS

Adapt the provided content for social media platforms.

## Step 1: Analyze Source Content

1. Read and understand the source material
2. Identify key insights and takeaways
3. Determine the core message
4. Find quotable/shareable elements

## Step 2: Create Platform-Specific Content

Create adapted versions for:

### Twitter/X
- Thread format (for detailed content)
- Single tweet (for quick shares)
- Consider hooks that work standalone

### LinkedIn
- Professional angle
- Personal story integration
- Clear takeaways

### Instagram (if visual content is relevant)
- Carousel concept
- Caption copy

## Step 3: Prepare Engagement Strategy

- Suggested hashtags per platform
- Best posting times
- Follow-up content ideas
