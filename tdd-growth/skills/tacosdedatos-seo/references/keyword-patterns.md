# Spanish Tech Keyword Patterns

Search behavior patterns for Spanish-speaking technical audiences.

## Mixed-Language Search Reality

Spanish developers search in hybrid patterns:

| Pattern | Example | Volume |
|---------|---------|--------|
| English tool + Spanish | "pandas tutorial español" | Higher |
| Spanish question + English | "cómo usar TensorFlow" | Higher |
| Full Spanish | "tutorial de ciencia de datos" | Medium |
| Full English | "pandas tutorial" | Highest (but more competition) |

## Optimal Target Keywords

Best ROI: Spanish phrase + English tool name + Spanish modifier

```
Primary:   "tutorial de pandas en español"
Secondary: "pandas tutorial español"
Long-tail: "cómo leer csv con pandas"
```

## Question Patterns

Spanish searchers use:
- "qué es [concept]" — what is
- "cómo [action]" — how to
- "para qué sirve [tool]" — what is X for
- "diferencia entre [X] y [Y]" — difference between
