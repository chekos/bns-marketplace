# Spanish Regional Terminology

Mexico vs Spain vs LATAM terminology for SEO.

## Technology Terms

| English | Mexico/LATAM | Spain | Use |
|---------|--------------|-------|-----|
| Computer | computadora | ordenador | **computadora** |
| Cell phone | celular | móvil | **celular** |
| Laptop | laptop | portátil | **laptop** |
| Mouse | mouse | ratón | **mouse** |
| File | archivo | fichero | **archivo** |
| USB drive | USB/memoria | pendrive | **USB** |

## Software Terms

| English | Mexico/LATAM | Spain | Use |
|---------|--------------|-------|-----|
| Application | aplicación | app | **aplicación** |
| Website | sitio web | web | **sitio web** |
| Click | clic | clic | **clic** |
| Link | enlace | enlace | **enlace** |
| Email | correo | email | **correo** |
| Upload | subir | subir | **subir** |

## Programming Terms

Keep in English across all regions:
- Python, JavaScript, Java, SQL, R
- pandas, numpy, scikit-learn, TensorFlow
- API, SDK, CLI, IDE
- machine learning, deep learning
- backend, frontend, framework

Translate these:
- code → código
- function → función
- variable → variable
- library → librería (Mexico) / biblioteca (Spain) → **librería**
- database → base de datos

## Pronoun Usage

| Context | Mexico/LATAM | Spain |
|---------|--------------|-------|
| Informal "you" (singular) | tú | tú |
| Informal "you" (plural) | **ustedes** | vosotros |
| Formal "you" | usted | usted |

**Rule**: Never use "vosotros" — it marks content as Spain-only.
