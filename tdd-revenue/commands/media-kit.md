When `/media-kit` is used with:

$ARGUMENTS

Generate or update the media kit for sponsor outreach.

## Step 1: Gather Current Metrics

Collect the latest data:
- Subscriber count (free and paid)
- Open rate (last 30 days average)
- Click-through rate (last 30 days average)
- Audience demographics (from surveys or analytics)
- Growth rate (month-over-month)

If specific numbers aren't provided, use placeholders that clearly need updating.

## Step 2: Compile Sponsorship History

If available, gather:
- Past sponsor names and results
- Average CTR for sponsored content
- Testimonials from sponsors
- Case studies

## Step 3: Structure the Media Kit

Create a comprehensive media kit following this structure:

### Required Sections
1. **Newsletter Overview** - Mission, voice, frequency
2. **Audience Profile** - Demographics, roles, interests
3. **Key Metrics** - Subscribers, engagement rates
4. **Sponsorship Options** - Placements, pricing, specs
5. **Case Studies** - Results from past sponsors
6. **Contact & Booking** - How to get started

## Customization Notes

If the user specifies:
- **"update metrics"**: Focus on the numbers sections
- **"add case study [company]"**: Add or update case study section
- **"for [specific sponsor]"**: Customize the positioning for that prospect
- **"Spanish version"**: Create in Spanish with appropriate localization
